# demo_cloud
○ Designed and developed “Cloud Cast” – a weather forecasting web application using HTML, CSS, and 
JavaScript with a clean, responsive UI. ○ Fetched and integrated real-time weather data from a public API to display live temperature, humidity, 
pressure, wind speed, and visibility. ○ Implemented features like today’s highlights, hourly temperature trends, sunrise & sunset timings, and 
5-day weather forecasts. ○ Built a fully dynamic frontend backed by JavaScript logic to update UI elements based on API data, 
enhancing interactivity and user experience
